# Online Shoppers Purchasing Intention

Predict whether an online shopper will make a purchase during a session using behavioral and traffic features.

## Dataset

`data.csv` contains session-level attributes (pageviews, durations, referral sources, visitor type, etc.) and the target column `Revenue` indicating purchase intent. It is taken from the [UC Irvine Machine Learning Repository](https://archive.ics.uci.edu/dataset/468/online+shoppers+purchasing+intention+dataset). 

Introductory paper: Sakar, C.O., Polat, S.O., Katircioglu, M. et al. Real-time prediction of online shoppers’ purchasing intention using multilayer perceptron and LSTM recurrent neural networks. *Neural Comput & Applic* 31, 6893–6908 (2019). https://doi.org/10.1007/s00521-018-3523-0

**Files**
- **data.csv:** Raw dataset with session features and the `Revenue` target.
- **notebook.ipynb:** Analyses and modelling pipeline (EDA, preprocessing, feature engineering, model training, evaluation, and conclusions).
